# RANDOM-DNA-SEQUENCE-GENERATOR-TOOL-AND-VISUALIZE-BASE-COMPOSITION-

DNA stands for Deoxyribonucleic Acid. It has a double helix structure. The sides of the ladder are a sugar-phosphate backbone.The helix is made up of four chemical bases: adenine (A), guanine (G), cytosine (C), and thymine (T), where A always pairs with T, and C always pairs with G.
